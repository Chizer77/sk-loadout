---
name: sk-save
description: Save current state as a preset. Use when user asks to save, snapshot, or create a preset or profile.
---

sk-loadout manages presets (model + skills). Run `sk codex ls` to see current state.

- Update current: `sk codex save`
- Create new: `sk codex save <name> --desc "<description>"`
- Overwrite existing: same as create
