---
name: sk-use
description: Switch to a preset. Use when user asks to switch, change, or activate a preset, mode, or profile.
---

sk-loadout manages presets (model + skills). Run `sk codex ls` to see available presets.

- Name given: `sk codex use "<name>"`
- No name: run `sk codex ls --json`, let user pick, then `sk codex use "<chosen>"`
