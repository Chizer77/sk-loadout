---
name: sk-add
description: Mount skills to current loadout. Use when user asks to add, mount, or install skills.
---

sk-loadout manages presets (model + skills) for OpenCode. Run `sk opencode ls` to see current state.

- Run `sk opencode ls` to list available skills
- Match names and run `sk opencode add <skill1> <skill2> ...`
- If a skill is not in the list, tell the user
