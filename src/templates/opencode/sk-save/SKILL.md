---
name: sk-save
description: Save current state as a preset. Use when user asks to save, snapshot, or create a preset or profile.
---

sk-loadout manages presets (model + skills) for OpenCode. Run `sk opencode ls` to see current state.

- Update current: `sk opencode save`
- Create new: `sk opencode save <name> --desc "<description>"`
- Overwrite existing: same as create
